## 実装サマリー

### 世界観
「裏路地に灯る、母ちゃんの台所。」をコアコンセプトに、あたたかく家庭的なシネマティック居酒屋サイトを実現。junni.co.jp から着想した「余白と写真のバランス」「日本語タイポグラフィの美しさ」「ストーリーテリング構成」を、BAR WARPとは対照的な「じわっと灯る」演出で実装。

### シネマティック演出

| 技法 | 実装内容 |
|---|---|
| ローダー | 「郁」の字が緩やかにclip-pathで出現 → フェードアウト（最低2200ms） |
| カスタムカーソル | 提灯の橙カラー・ring 36px・lag 0.10でふわっと追従 |
| h1 clip-pathリビール | 「裏路地に灯る、」「母ちゃんの台所。」が幕が上がるように出現 |
| ワードスタガー | eyebrowが単語ごとに時差出現 |
| スクロールリビール | [data-warp] + [data-fade] で全セクション展開 |
| 水平スクロール | 利用シーン5カードをドラッグ操作 |
